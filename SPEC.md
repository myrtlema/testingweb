# 中文轉英文翻譯網站規格文檔

## 1. 專案概述

- **專案名稱**: Chinese to English Translator
- **專案類型**: 單頁翻譯工具網站
- **核心功能**: 讓使用者輸入中文文字，即時翻譯成英文
- **目標用戶**: 需要中英翻譯的使用者

## 2. UI/UX 規格

### 2.1 版面配置

- **Header**: 網站標題和簡潔的導航
- **主要區域**: 雙向文字輸入區塊（中文輸入 / 英文輸出）
- **功能區**: 翻譯按鈕和輔助功能
- **Footer**: 版權資訊

### 2.2 響應式斷點

- **桌面**: >= 768px（雙欄並排）
- **手機**: < 768px（單欄堆疊）

### 2.3 視覺設計

#### 色彩方案
- **主色**: #1a1a2e (深藍黑)
- **次色**: #16213e (深藍)
- **強調色**: #e94560 (珊瑚紅)
- **背景色**: #0f0f1a (近黑)
- **文字色**: #eaeaea (淺灰白)
- **次要文字**: #a0a0a0 (灰色)
- **卡片背景**: #1f1f3a (深紫藍)
- **輸入框背景**: #252545 (深灰藍)

#### 字體
- **標題字體**: "Noto Sans TC", "Playfair Display", sans-serif
- **內文字體**: "Noto Sans TC", system-ui, sans-serif
- **標題大小**: 2.5rem (h1), 1.5rem (h2)
- **內文大小**: 1rem
- **輸入文字大小**: 1.1rem

#### 間距系統
- **區塊間距**: 2rem
- **內邊距**: 1.5rem
- **元素間距**: 1rem

#### 視覺效果
- **卡片陰影**: 0 8px 32px rgba(233, 69, 96, 0.1)
- **按鈕懸停**: 發光效果 box-shadow: 0 0 20px rgba(233, 69, 96, 0.5)
- **輸入框焦點**: 邊框發光效果
- **動畫過渡**: 0.3s ease

### 2.4 元件設計

#### 翻譯區域
- 兩個大型文字區域並排
- 左側：中文輸入（可編輯）
- 右側：英文輸出（唯讀）
- 支援拖曳調整大小
- 字元計數顯示

#### 功能按鈕
- **翻譯按鈕**: 珊瑚紅色，懸停發光
- **清除按鈕**: 次要樣式
- **複製按鈕**: 輸出區域專用
- **交換按鈕**: 交換輸入輸出位置

#### 狀態指示
- 翻譯中顯示載入動畫
- 翻譯完成顯示成功提示

## 3. 功能規格

### 3.1 核心功能

1. **中文輸入**
   - 文字輸入框，支援多行輸入
   - 即時顯示字元數
   - 最大字元限制：5000

2. **翻譯功能**
   - 點擊翻譯按鈕執行翻譯
   - 使用翻譯 API 進行翻譯
   - 顯示翻譯結果在輸出區域

3. **清除功能**
   - 一鍵清除輸入和輸出

4. **複製功能**
   - 一鍵複製翻譯結果到剪貼簿

5. **交換功能**
   - 交換輸入和輸出內容

### 3.2 使用者互動流程

1. 使用者在左側輸入中文
2. 點擊「翻譯」按鈕
3. 系統顯示載入中狀態
4. 翻譯完成，顯示英文結果
5. 使用者可複製結果或繼續輸入

### 3.3 邊界情況處理

- 空輸入：顯示提示訊息
- 翻譯失敗：顯示錯誤訊息
- 網路錯誤：顯示重試提示

## 4. 技術實現

- **HTML5**: 語意化標籤結構
- **CSS3**: 自定義屬性、動畫、響應式設計
- **JavaScript**: 原生 JS，無框架依賴
- **翻譯 API**: MyMemory Translation API (免費 API)
